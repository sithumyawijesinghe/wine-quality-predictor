Wine Quality Predictor App

This project is a machine learning-powered web app built with Streamlit that predicts the quality of wine based on
various chemical properties.

Features

- Simple and user-friendly interface.  
- Predicts wine quality using key features like acidity, sugar, alcohol content, and more.  
- Inputs accepted via sidebar sliders and number inputs.  
- Displays predicted wine quality score immediately.

Technologies Used

- Python (pandas, scikit-learn, pickle)  
- Streamlit for building the web app  
- GitHub for version control  
- Streamlit Cloud (optional) for easy deployment
